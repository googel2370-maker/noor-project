Noor iOS Skeleton
-----------------
Files included:
- AppConfig.swift  (set your server URL)
- ChatMessage.swift
- NoorPersona.swift (system prompt - edit to customize Noor)
- KeychainHelper.swift
- PermissionManager.swift
- PushManager.swift
- NetworkManager.swift
- ActionApprovalView.swift
- ContentView.swift

How to use:
1. Open Xcode, create a SwiftUI project, and add these files.
2. Set your AppConfig.serverBase to your server URL.
3. Configure Signing & Capabilities: Push Notifications.
4. Run on your device to test.
